CS50 Web Development with Python and Javascript

Over the winter I had made it my mission to complete the CS50 web development course. I found that Harvard’s CS50 course was more than your usual YouTube tutorial! This class forces the student to learn on their own, think creatively and come up with their own ideas. For example project, one has a list of specifications that your program must meet. The specs cover everything from user logins, database search query’s, AJAX, API’s, version control and much much more! At the time of this writing, I have not completed the course. The following is a status report on my progress so far.

I can say for certain that learned the following:

  • Basic version control in git - branching, stashing, pushing to the origin, making a pull request
  • Connecting to a remote database (Heroku)
  • Basic JavaScript DOM manipulation
  • Basic AJAX requests with javascript
  • Basic user authentication
  • Why web applications are so vulnerable

Skills that I improved upon:

  • Managing a Model View Template based web application
  • Efficient folder structure
  • Making meaningful commit messages

What I still need to learn:

  • Creating a proper Github Read Me
  • Using JSON data from a XMLHttpRequest to create HTML elements
  • Creating an API with Flask
  • Passing data between routes in a MVT web application
  • Better (and more secure) database interaction.

In Summary

The “what I still need to learn” column is far from complete. I have a lot to learn and I am very excited to do so. As for my learning path, I have to say that although I am not following a set path, I have taken a great deal of advice from the p1xt programming curriculum. Stay tuned, there is more to come!
